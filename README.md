# AIFFEL_quest_cr

## 레포지토리 소개
이 레포지토리는 AIFFEL 코어 과정을 진행하며 생성된 프로젝트와 학습 자료를 포함합니다.
폴더는 각 퀘스트와 탐구 주제에 따라 구성되어 있으며, 퀘스트별 작업 내용과 참고 자료를 정리하였습니다.
